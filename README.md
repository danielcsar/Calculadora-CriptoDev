## Como Rodar

### Pré-requisitos

- [Nodejs](https://nodejs.org/en/) (v16 ou superior)

### Rodando o projeto

Na raiz do projeto execute os seguintes comandos:

```bash
# Instalando dependências
$ npm install
# Rodando o projetos
$ npm run dev
```

## Biblioteca utilizada

- Tailwindcss - [TailwindCss](https://tailwindcss.com/)
